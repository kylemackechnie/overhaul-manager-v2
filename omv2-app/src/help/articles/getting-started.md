# Getting Started

Welcome to **Overhaul Manager**. This guide gets you from zero to your first project in a few minutes.

## What this app is for

Overhaul Manager tracks the cost and execution of turbine outage projects — timesheets, expenses, equipment hire, purchase orders, invoices, and reconciliation back to SAP. It's built for project managers and engineers running steam and gas turbine overhauls.

## The three things you need to know first

**1. Everything is project-scoped.** You always have one active project, shown in the top bar. Switch projects from the picker in the header. Cost data, timesheets, resources — all of it belongs to one project at a time.

**2. The ribbon is your navigation.** Tabs across the top group features by area (Project, Cost Tracking, Personnel, etc.). Clicking a tab swaps the buttons below — clicking a button takes you to that panel.

**3. The File menu has the global stuff.** The purple **File** button on the far left of the ribbon opens a dropdown for things that aren't project-specific: user management, audit trail, backups, and this Help & Guide.

## Your first 10 minutes

If you're brand new, do these in order:

1. **Open Project Settings** (⚙ Settings in the Project tab) — set the project name, dates, base currency, and exchange rates.
2. **Add WBS codes** (Cost Tracking → WBS) — these underpin every cost line.
3. **Add resources** (Personnel → Resources) — the people on the job.
4. **Set up rate cards** (Personnel → Rate Cards) — hourly rates per role.
5. **Create your first timesheet week** (Personnel → Timesheets) — pick the week, add crew, enter hours.

That's the minimum viable setup. Everything else (POs, invoices, TCE, hire, tooling) builds on top.
